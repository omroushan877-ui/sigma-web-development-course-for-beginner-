# Sigma Web Development Course for Beginners

A comprehensive beginner-friendly web development course that takes you from HTML basics to interactive JavaScript projects.

## 📚 Course Overview

This course is designed for complete beginners who want to learn web development from the ground up. Through 17+ practical video lessons, you'll master the fundamentals of **HTML**, **CSS**, and **JavaScript** and build real projects along the way.

## 🎯 What You'll Learn

- **HTML Fundamentals**: Structure, semantic HTML, forms, and multimedia elements
- **CSS Styling**: Layout, flexbox, positioning, and responsive design
- **JavaScript Basics**: Variables, functions, DOM manipulation, and event handling
- **Interactive Web Projects**: Build functional applications combining HTML, CSS, and JavaScript

## 🚀 Getting Started

### Prerequisites
- A code editor (VS Code, Sublime Text, etc.)
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic computer skills

### How to Use This Repository

1. **Clone or download** this repository to your local machine
2. **Navigate** to any `video [number]/` folder
3. **Open** `index.html` in your web browser to see the project
4. **Review** the `script.js` and `style.css` files to understand the code
5. **Experiment** by modifying the code and refreshing the browser to see changes

### File Structure

Each video folder contains:
- `index.html` - The HTML structure
- `style.css` - CSS styling (when applicable)
- `script.js` - JavaScript code (when applicable)

## 💡 Learning Tips

- **Don't Skip**: Watch each video before reviewing the code
