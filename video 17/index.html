CSS selectors

CSS selectors are used to select the HTML elements that we want to style.

There are different types of CSS selectors, such as:

  • Element selector
  • Class selector
  • ID selector
  • Attribute selector
  • Pseudo-class selector
  • Pseudo-element selector 
        /* CSS selector syntax */
        selector {
            property: value;
        }

        /* ID selector */
        #id {
            property: value;
        }

        /* Class selector */
        .class {
            property: value;
        }

        /* Element selector */
        element {
            property: value;
        }
